# kata-python

## Setup

Simply run ```./install/sh```.

## Problem
```
String Sum

Objetivo: Somar todos os números inteiros de uma string

1. "" resulta 0
    "" => 0

2. Deve receber e somar números separados por string.
    "3" => 3
    "2,3" => 5
    "2,3,4" => 9

3. Deve aceitar também quebras de linhas
    "1\n2" => 3
    "1\n2,3" => 6

4. Deve lançar uma exceção ao encontrar números negativos: 
    "2,-1" => "negative numbers not allowed: -1"

5. Deve mostrar na exceção todos os números negativos encontrados: 
    "2,-1,-2" => "negative numbers not allowed: -1, -2"

6. Deve aceitar outros separadores, como ";" e " " (espaço).

```
